T-n-T Inventory Management System

Tudor Minut and Tudor Tugui

2025-06-28

Abstract

This report describes the T-n-T web application for inventory management, including features for administration, notifications, import/export, and statistics. The system is designed with security and usability in mind, using modern web technologies and a PostgreSQL database.

Introduction

The T-n-T application is a web-based platform for managing items, consumables, and spare parts. It provides a complete set of features for user authentication, category and item management, notifications, and data analysis.

Technologies Used

  • Backend: TypeScript with native Node.js HTTP module (no frameworks)
  • Frontend: HTML, CSS, Vanilla JavaScript/TypeScript
  • Database: PostgreSQL
  • Authentication: bcrypt for password hashing

Security Features

SQL Injection Prevention

  • All queries use prepared parameters ($1, $2, etc.)
  • Type validation for all inputs
  • Sanitization and validation of all input data

Cross-Site Scripting (XSS) Prevention

  • Automatic sanitization of all HTML outputs
  • Use of textContent instead of innerHTML in the frontend
  • Implementation of a sanitizeHtml() function for all displayed data

Other Security Measures

  • Password hashing with bcrypt and salt
  • Comprehensive validation of emails, passwords, and data
  • Security headers: X-Content-Type-Options, X-Frame-Options, X-XSS-Protection
  • Configurable CORS for production and development
  • Input length validation to prevent overflows

Features

  • User authentication and authorization
  • CRUD management for categories and items
  • Notification system for low stock
  • Data import/export (CSV, JSON, XML)
  • Statistics generation (HTML, PDF)
  • Admin panel for user management
  • Security measures against common attacks

Application Usage

  1. Start the server: npm run dev in the backend directory
  2. Access the application at http://localhost:3000
  3. Create the admin account: node createAdmin.js

Security and Compliance

This application implements robust security measures in line with industry standards to prevent common web vulnerabilities (OWASP Top 10).
